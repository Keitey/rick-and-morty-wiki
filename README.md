# Rick & Morty Wiki

Este é uma wiki do desenho Rick & Morty feita em <strong>React</strong> usando <strong>Bootstrap</strong> e <strong>SASS</strong> consumindo a API oficial, para construção dele foram usados:

* Cards;
* Font size;
* Imagens responsivas;
* Fecth;
* React Paginate;
* Dinamic pages;
* Layout responsivo;
* Accordion;
* Grid system;
* react-router-dom;

## Veja o projeto completo:

https://wiki-and-morty-wiki.netlify.app/
